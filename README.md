# 🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.
The game allows the user to play against the computer in real time.

---

## 🚀 Features

* 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
* 🤖 Computer generates random moves
* 🏆 Instant result display (Win / Lose / Draw)
* 🎯 Simple and responsive UI
* ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

---

## 🎮 How to Play

1. Click on Rock, Paper, or Scissors
2. Computer will automatically choose its move
3. Result will be displayed instantly
4. Play again to improve your score!
